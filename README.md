
# 🏋️ Gym Members Data Analysis | Excel, SQL & Power BI

## 📌 Project Overview

This project analyzes a Gym Members dataset using **Microsoft Excel, MySQL, and Power BI** to uncover insights into workout performance, member demographics, and fitness trends. The analysis follows a complete data analytics workflow, including data cleaning, exploratory analysis, SQL-based business insights, and interactive dashboard creation.

The objective is to transform raw fitness data into meaningful insights that help understand workout effectiveness, calorie expenditure, and member behavior.

---

## 🎯 Objectives

* Clean and prepare the dataset for analysis.
* Analyze workout and fitness trends.
* Identify factors affecting calorie burn.
* Compare member performance across age groups, gender, and experience levels.
* Build an interactive Power BI dashboard for data-driven decision-making.

## 📂 Project Workflow

### 1. Data Cleaning (Excel)

* Removed duplicate records.
* Checked and handled missing values.
* Corrected data types and formatting.
* Validated dataset consistency.

### 2. SQL Analysis

Business questions answered include:

* Which age group burns the most calories?
* Which workout type burns the most calories?
* Which workout type has the highest average heart rate?
* Which experience level burns the most calories?
* Does longer workout duration burn more calories?
* Which gender burns more calories on average?

### 3. Power BI Dashboard

Created an interactive dashboard featuring:

* KPI Cards
* Workout Analysis
* Calories Burned Analysis
* Heart Rate Analysis
* Gender Comparison
* Age Group Analysis
* Experience Level Analysis
* Interactive Slicers

---

## 📊 Dashboard KPIs

* Total Members
* Average Calories Burned
* Average Workout Duration
* Average Heart Rate (BPM)
* Average BMI
* Average Water Intake

---

## 🔍 Key Insights

* Workout type significantly influences calorie expenditure and workout intensity.
* Longer workout sessions generally result in higher calorie burn.
* Experienced members tend to burn more calories due to higher workout intensity.
* Heart rate varies across workout types, reflecting differences in exercise intensity.
* Calorie expenditure differs among age groups and genders.
* Interactive dashboards make it easier to identify workout trends and compare member performance.

---

## 💼 Business Value

The insights from this analysis can help:

* Design personalized workout plans.
* Recommend effective workout programs.
* Monitor workout intensity using heart rate.
* Improve member engagement and fitness outcomes.
* Support data-driven decision-making in fitness centers.
